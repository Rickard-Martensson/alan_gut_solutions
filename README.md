# Alan Gut Solutions Project

## Project Overview
This repository contains solutions to problems found in `An intemedate Course in Probability`, one of the best books I've ever read! This project aims to provide detailed solutions and explanations for the exercises in the book to assist students and educators in mastering the material.

## Installation
To set up this project locally, run the following commands:
```
git clone https://github.com/<your-username>/alan_gut_solutions
cd alan_gut_solutions
```

## Usage
To compile a LaTeX file to a PDF:
```
pdflatex Chapter1.tex
```
To execute a Python script:
```
python ex3_4_8.py
```
